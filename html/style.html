Style Attribute in HTML


~The style attribute specifies an inline style for an element.

~The style attribute will override any style set globally.

~Inline Style: In Inline styling, the CSS rules are directly written inside the starting tag using the style attribute.

~The style attribute includes a series of CSS property and value pairs. Each ‘ property: value ‘ pair is separated by a semicolon ( ; ).

~This attribute will override the style properties globally for any relevant style set.
